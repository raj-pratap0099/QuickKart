🛒 QuickKart - MERN E-Commerce Platform

QuickKart is a full-stack e-commerce web application built using the MERN stack (MongoDB, Express, React, Node.js). It includes a user-facing store, 
admin dashboard, authentication, cart, orders, and payment integration.

🚀 Live Demo
🌐 Frontend: https://quickkart-frontend-cdi7.onrender.com/
🛠 Admin Panel: https://quickkart-admin-qe91.onrender.com

✨ Features
🔐 User Authentication (JWT + Cookies)
🛍 Product Listing & Filtering
🛒 Cart Management
📦 Order Placement
💳 Razorpay Payment Integration
☁️ Image Upload (Cloudinary)
🧑‍💼 Admin Dashboard (Manage Products & Orders)
🎙 AI Voice Assistant (Speech Recognition)

🏗 Tech Stack
Frontend: React, Tailwind CSS
Backend: Node.js, Express.js
Database: MongoDB
Auth: JWT + Cookies
Cloud: Cloudinary
Payments: Razorpay

⚙️ Environment Variables

📦 Installation
# Clone repository
git clone https://github.com/raj-pratap0099/QuickKart.git

# Go to project folder
cd QuickKart
# Install backend dependencies
npm install
# Install frontend dependencies
cd frontend
npm install
▶️ Run Locally
# Run backend
npm run server
# Run frontend
npm run dev
